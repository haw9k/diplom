# План кодирования этапов A1-A7

## A1
Сначала реализуется инфраструктура источников, клиентов API и инкрементальная синхронизация с локальным хранилищем.

## A2
Потом добавляются нормализация строк, дедупликация и правила верификации работодателей.

## A3
Дальше подключаются пайплайны признаков, NER и классификация записей о работе.

## A4
После этого собираются метрики, пороги качества и решение о публикации модели.

## A5
Затем оформляются артефакты модели, docker-сборка и проверка готовности сервиса.

## A6
Далее готовятся агрегаты для интерфейса и выдача данных в шаблоны.

## A7
В конце добавляются метрики, журналы, алерты и сигналы повторного запуска этапов.
